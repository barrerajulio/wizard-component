## React Component for rendering a wizard

A basic wizard that renders a big page in small steps

### Component usage

```js
<Wizard>
  <Wizard.Header>
    <h1>Header</h1>
  </Wizard.Header>
  <Wizard.Body>
    <Wizard.Content>...</Wizard.Content>
    <Wizard.Content>...</Wizard.Content>
  </Wizard.Body>
  <Wizard.Footer></Wizard.Footer>
</Wizard>
```

### Using props to customize the wizard

Modify the content to see it change live:

```js live
<Wizard></Wizard>
```
